Билет на твой первый концерт в Лужниках содержит 8 цифр от 0 до 3. 
Чтобы увеличить прибыль, перекупщики придумали понятие обобщенно-счастливого билета. 
Обобщенно-счастливым называется билет, номер которого можно разбить на две части таким образом, 
чтобы сумма цифр обеих частей совпадала.
